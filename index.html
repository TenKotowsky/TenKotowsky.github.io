LuaBot

LuaBot is a multi-purpose Discord bot equipped with moderation tools, community managment tools and other fun features!



An example of what you can do with it

You can equip your server with automated QOTD/QOTW, get some gaming-related info with gaming commands or have fun with 4FUN commands! Get a random quote, a recipe for a dish, see info about a Roblox user, enlarge emojis - you name it!
